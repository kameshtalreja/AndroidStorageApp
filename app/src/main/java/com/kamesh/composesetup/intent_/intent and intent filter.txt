
intent : is messing object which tells that what kind of action be performed. the intent's most significant job is
to launching activity


implicit intent : doesn't specify the component, in such a case intent provide information on available components
provided by the system that is to be invoked.

ex : Intent(Intent.ACTION_VIEW) intent.setData(Uri.parse("url.com")) you can write URL

explicit intent : specify the component. in such a case, intent provides the external class to be invoked.
